# Funciones en C# - Scripting

Este proyecto corresponde a una actividad académica de la materia de scripting.

## Descripción
El programa contiene 20 firmas diferentes de funciones en C#, con sus respectivas invocaciones.
Se incluyen funciones:
- Con y sin parámetros
- Con y sin retorno
- Uso de distintos tipos de datos como int, string, bool, arreglos y clases
